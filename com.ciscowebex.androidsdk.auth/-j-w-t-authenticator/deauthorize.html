deauthorize

fun deauthorize(handler: CompletionHandler<Void>?): Unit

Overrides Authenticator.deauthorize

Deauthorizes the current user and clears any persistent state with regards to the current user. If the com.ciscowebex.androidsdk.phone.Phone is registered, it should be deregistered before calling this method.

After this method has been invoked, for continued usage, new instances of Authenticator and Webex need to be created and used
