setVideoMaxRxBandwidth

abstract fun setVideoMaxRxBandwidth(bandwidth: Int): Unit

Set the max receiving bandwidth for video in unit bps for the call. Only effective if set before the start of call. if 0, default value of 2000*1000 is used.

Parameters

bandwidth - the suggest value could be DefaultBandwidth.MAX_BANDWIDTH_90P, DefaultBandwidth.MAX_BANDWIDTH_180P, DefaultBandwidth.MAX_BANDWIDTH_360P, DefaultBandwidth.MAX_BANDWIDTH_720P, or DefaultBandwidth.MAX_BANDWIDTH_1080P.
