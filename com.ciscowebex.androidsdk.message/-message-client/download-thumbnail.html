downloadThumbnail

abstract fun downloadThumbnail(remoteFile: RemoteFile, path: File?, completionHandler: CompletionHandler<Uri>): Unit

Downloads the thumbnail (preview image) of a file attachment asynchronously. Note Cisco Webex doesn't generate thumbnail for all files. See File Sharing for more details.

Parameters

remoteFile - The remote file whose thumbnail to be downloaded.

path - The local file directory to save the thumbnail.

completionHandler - A closure to be executed when the download has completed. The URI contains the path to the downloaded thumbnail.
