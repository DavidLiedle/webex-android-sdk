AdvancedSetting

abstract class AdvancedSetting<T>

Types

ShareMaxCaptureFPS

 class ShareMaxCaptureFPS : AdvancedSetting<Int>

Set the max screen capture fps for screen share for the call. If 0, default value of 5 is used. Range of this value is 1 to 10 If set the value >5 and battery / CPU are not acceptable on the device, then should set the value <= 5.

VideoEnableCamera2

 class VideoEnableCamera2 : AdvancedSetting<Boolean>

Use android.hardware.camera2.CameraDevice or use android.hardware.Camera. The default is camera2. This setting is invalid on Android API 21.

VideoEnableDecoderMosaic

 class VideoEnableDecoderMosaic : AdvancedSetting<Boolean>

Enable or disable the video mosaic for error-concealment when data loss in network. The default is enable.

VideoMaxTxFPS

 class VideoMaxTxFPS : AdvancedSetting<Int>

Set the max sending fps for video for the call. If 0, default value of 30 is used.
